# fullStack-exercise
The next exercise is an evaluation test.

It is made up of a client in reactjs and an APi in Nodejs & express, with tests with the supertest library.
It contains several components and a great variety of resources of this frameworks(React and Express).

The client requests the weather information through the developed API and it makes the corresponding API requests from the Open Weather Map.
Api: port 4000 Nodejs & express
Client: 3000 react project
